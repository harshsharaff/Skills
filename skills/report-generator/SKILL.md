# Report Generator

## Instructions

1. Clarify the subject and audience if not given.
2. Fill in [templates/report.md](templates/report.md), keeping every section.
3. Lead with the executive summary; support each finding with data.
4. Keep recommendations specific and actionable.

For section-by-section guidance and tone rules, see [reference.md](reference.md).

## Output

Always emit the completed report using the template structure — do not invent
new top-level sections.
